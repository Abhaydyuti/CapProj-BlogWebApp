# Blog Web Application

A full-stack blog platform built with Node.js, Express.js and EJS templating.

## About
A server-side rendered blog application where users can create and read blog posts. Built as a capstone project demonstrating full-stack web development skills.

## Technologies Used
- Node.js
- Express.js
- EJS (Embedded JavaScript Templating)
- HTML5
- CSS3

## Features
- Dynamic server-side rendering with EJS
- RESTful routes for blog post management
- Responsive UI
- Create and view blog posts

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `node index.js`
4. Open `http://localhost:3000` in your browser
